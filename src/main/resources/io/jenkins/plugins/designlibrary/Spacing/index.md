# Spacing

> Defines consistent padding, margins, and gaps between elements.

**Category:** Patterns

Jenkins features a range of spacing helper classes to enable you to set the margin and padding for your element.

This is done through the `jenkins-!-margin-*` and `jenkins-!-padding-*` classes. The `*` is the spacing modifier, it ranges from `0` for no spacing to `6` for maximum spacing.

You can use position modifiers if you only want to impact one side, for example, `jenkins-!-padding-left-3`.
