# Banner

> Banners are a handy way to display small snippets of text and actions.

**Category:** Components

## Info

### info

```xml
<div class="jenkins-alert jenkins-alert-info">
  Hello world
</div>
```

## Warning

### warning

```xml
<div class="jenkins-alert jenkins-alert-warning">
  Hello world
</div>
```

## Danger

### danger

```xml
<div class="jenkins-alert jenkins-alert-danger">
  Hello world
</div>
```

## Success

### success

```xml
<div class="jenkins-alert jenkins-alert-success">
  Hello world
</div>
```

> Since 2.459. The css classes have the jenkins- prefix. Older versions should use the non prefixed form which interfere with classes defined in bootstrap.
